### Этап 1: MVP (Ручной режим, 3-5 дней)
**Цель:** Получить первые лиды и подтвердить связку, используя Ваш "чистый" испанский траст.

1.  **Инфраструктура:**
    *   *Телефон:* Ваш iPhone/Android с испанской SIM (Wi-Fi можно, но лучше раздавать 4G/5G, чтобы IP был мобильным).
    *   *Прокси:* Не нужны (Вы уже резидент).
    *   *Софт:* CapCut (мобильный), Telegram (для связи с партнеркой).

2.  **Офферы (Куда лить):**
    Для Испании (Tier-1) и быстрого старта рекомендую следующие CPA-сети. Регистрируйтесь и сразу пишите менеджеру: "I have organic TikTok traffic from Spain, looking for smartlink or best offers".
    *   **iGaming / Gambling (Высокий риск/доход):**
        *   *LGaming / 3SNET:* Отличные офферы под Европу. Ищите офферы типа "VegasHero" или локальные испанские казино. Выплата за депозит (CPA) в Испании может достигать $150-250.
        *   *Схема:* Видео с эмоциями выигрыша -> Ссылка в профиле на Telegram-канал -> В канале "прогрев" и ссылка на казино.
    *   **Dating / Adult (Стабильно):**
        *   *LosPollos / Traforce / iMonetizeIt:* Это смартлинки. Вы просто кидаете одну ссылку, а система сама подбирает оффер под пользователя. Для старта идеально.
        *   *Схема:* Видео с красивыми девушками (не 18+, просто attractive) -> Ссылка в профиле "See more here" (через прокладку) -> Дейтинг сайт.
    *   **Telegram-каналы (Безопасно):**
        *   Биржи типа *Telega.in* (есть зарубежные разделы) или прямой поиск админов испанских каналов. Это сложнее автоматизировать на старте, лучше взять CPA-оффер.

3.  **Ссылка (Техничка):**
    TikTok банит прямые партнерские ссылки.
    *   *Решение для MVP:* Используйте **Linktree** или сделайте простую HTML-страничку на **Github Pages** (Вы же разработчик), где будет одна кнопка "Enter".
    *   *Домен:* Купите дешевый домен .com/.net (не .ru!), привяжите к Github Pages/Cloudflare.

### Этап 2: Масштабирование (Automation & Cloud)
**Цель:** Создать конвейер. Заменить Вас скриптами и облачными сервисами.

1.  **Облачные телефоны (Cloud Phones) vs Эмуляторы:**
    *   *Важно:* TikTok в 2026 году моментально детектит обычные эмуляторы (Bluestacks, Nox) и серверные Android. Вам нужны **Real Device Cloud** или специализированные сервисы с подменой fingerprint.
    *   **Ресурсы:**
        *   **GeeLark / Redfinger:** Популярные облачные Android-телефоны. GeeLark позиционируется именно как "Anti-detect Cloud Phone" для TikTok. Он подменяет IMEI, MAC, модель.
        *   **VMOS Cloud:** Еще один вариант, позволяет создавать изолированные Android-среды.
        *   **Физическая ферма (Hardcore):** Если облака начнут банить, купите 5-10 б/у Xiaomi/Pixel, подключите их к USB-хабу и управляйте через **Scrcpy** + Python скрипты (ADB). Это самое надежное решение.

2.  **Прокси (Для масштаба):**
    Когда уйдете с домашнего Wi-Fi в облако, нужны **испанские мобильные прокси**.
    *   **iProxy.online:** Можно поднять свои прокси, если купите пару USB-модемов и SIM-карт в Испании. Это дешевле и качественнее покупных.
    *   **LTEBoost / Soax:** Если лень поднимать свои, покупайте мобильные 4G прокси (Испания). Ротация IP каждые 5-10 минут.

3.  **Content Production (Python + AI):**
    Вам нужно 100+ видео в день. Руками это делать нельзя.
    *   *Spy Tools:* **Minea** или **PiPiAds** (дорого, но топ). Скачивайте трендовые креативы конкурентов без водяных знаков (скрипт на Python + `yt-dlp` или API сервисов).
    *   *Уникализация (ffmpeg):* Напишите скрипт на Python, который берет скачанное видео и:
        *   Меняет MD5 хеш.
        *   Ускоряет видео на 2-3%.
        *   Накидывает невидимый шум (noise overlay).
        *   Отражает зеркально (если нет текста).
        *   Накладывает музыку из библиотеки TikTok (важно для траста).

4.  **Автоматизация залива (Playwright/Appium):**
    *   Для веб-версии TikTok: **Playwright** с плагином `stealth`. Но веб-версия дает меньше охватов (shadowban чаще).
    *   Для Android (лучший вариант): **Appium** или кликеры поверх **ADB**. Скрипт должен: Открыть TikTok -> Логин -> Загрузить видео из папки -> Написать теги -> Опубликовать.
